# 이미지 폴더

이 폴더에 웹사이트에 사용할 이미지를 업로드하세요.

## 필요한 이미지 목록

### 필수 이미지:
1. **hero-image.jpg** (메인 페이지 히어로 이미지)
   - 권장 크기: 800x600px
   - 교육 현장 또는 전문적인 이미지

2. **instructor.jpg** (강사 프로필 사진)
   - 권장 크기: 600x600px (정사각형)
   - 전문적인 프로필 사진

3. **course-1.jpg** (성희롱예방교육 이미지)
   - 권장 크기: 600x400px

4. **course-2.jpg** (직장 내 괴롭힘 예방교육 이미지)
   - 권장 크기: 600x400px

5. **course-3.jpg** (인권 감수성 교육 이미지)
   - 권장 크기: 600x400px

6. **course-4.jpg** (관리자 특화 교육 이미지)
   - 권장 크기: 600x400px

## 이미지 준비 가이드

### 파일 형식
- JPG 또는 PNG 사용
- 파일 크기는 각 이미지당 500KB 이하 권장

### 파일명 규칙
- ✅ 영문 소문자 사용
- ✅ 하이픈(-) 사용 가능
- ❌ 한글, 띄어쓰기, 특수문자 사용 금지
- ❌ 예: profile_photo.jpg (언더스코어 사용)
- ✅ 예: profile-photo.jpg (하이픈 사용)

### 이미지 최적화
무료 이미지 압축 사이트:
- https://www.iloveimg.com/ko/compress-image
- https://tinypng.com

### 무료 이미지 다운로드
- Unsplash: https://unsplash.com
- Pexels: https://www.pexels.com/ko-kr
- Pixabay: https://pixabay.com/ko

## 임시 이미지 사용

현재 이미지가 없는 경우:
1. 무료 이미지 사이트에서 적절한 이미지 다운로드
2. 이미지 크기 조정 및 압축
3. 위의 파일명으로 저장
4. 이 폴더에 업로드

## 이미지 교체 방법

### GitHub에서 교체:
1. GitHub Repository > images 폴더 클릭
2. **Add file** > **Upload files**
3. 새 이미지 드래그 & 드롭
4. 같은 파일명이면 자동으로 교체됨
5. **Commit changes** 클릭

### 주의사항:
- 이미지 교체 후 5-10분 대기 (GitHub Pages 반영 시간)
- 브라우저 캐시 삭제 (Ctrl+Shift+R)
